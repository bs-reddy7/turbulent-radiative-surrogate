# turbulent-radiative-surrogate

Surrogate model for turbulent radiative layer (TRL) simulations using Fourier Neural Operators (FNO).

## Quickstart

```bash
git clone https://github.com/<your-username>/turbulent-radiative-surrogate
cd turbulent-radiative-surrogate
pip install -r requirements.txt
python src/train.py --config configs/fno_trl2d.yaml
```

## Repository Structure

```
src/          training, evaluation, visualization scripts
configs/      YAML experiment configs
notebooks/    exploratory notebooks
project/      planning and architecture docs
```
